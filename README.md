# char-rnn-writer

# 训练结果：

**君**

君王安死处，教柯得仙容。
雾露老寒浦，风吹古角声

**山**

山色寒不高，松连气含渠。
暂行不见江，自属寒露冷。

**夜**

夜皆凤义花，浮云引艳切。
红进双柳赠，风雨万声和。


上面只是理想的测试结果，大家输入一些字之后会发现很多不理想的情况(～￣▽￣)～
